วิธีที่ 1 — Publish to web (แนะนำ, ทำครั้งเดียว):
1. เปิด Google Sheet (Database)
2. File → Share → Publish to web
3. เลือก sheet Database → เลือก Comma-separated values (.csv)
4. กด Publish → คัดลอก URL ทั้งหมด
5. วาง URL นั้นในช่อง Sheet ID/URL ด้านล่าง

วิธีที่ 2 — ใช้ Sheet ID ธรรมดา:
คัดลอก ID จาก URL: spreadsheets/d/SHEET_ID/edit
→ Sheet ต้อง Share เป็น Anyone with the link can view

• ไม่ต้อง Deploy, ไม่ต้อง Allow Permission
• Signal Date ถูกต้อง (ไม่ lag ซ้ำซ้อน)
• คำนวณใน browser โดยตรง เร็วกว่าเดิม

❓ EXIT_DD คืออะไร?
EXIT_DD = Emergency Exit เมื่อ Drawdown เกิน -6%

คือระบบป้องกันความเสี่ยง — ถ้ามูลค่าพอร์ตลดลงจาก Peak เกิน 6%
ระบบจะสั่ง ขายออกทั้งหมด → ย้ายเข้า Money_Market 100%

ตัวอย่าง: Peak = 130,000฿ → ถ้า Portfolio ลงมาที่ 122,200฿ (-6%) → EXIT

หลัง EXIT:
  • รอ 10 วันทำการ (Exit Cooldown)
  • รอให้ Signal กลับมาแข็งแกร่ง (score ≥ 0.55)
  • ถ้า EXIT 2ครั้งใน 30วัน → รอพักเพิ่มอีก (Extra Pause)
