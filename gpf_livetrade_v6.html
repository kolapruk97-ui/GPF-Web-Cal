วิธี Deploy GAS (ทำครั้งเดียว):

  1. เปิด Google Sheet ที่มีข้อมูล NAV
  2. เมนู Extensions (ส่วนขยาย) → Apps Script
  3. ลบ code เดิมทิ้งทั้งหมด → วาง code จากไฟล์ GPF_GAS_v3.js
  4. กด Save (Ctrl+S)
  5. ⚠️ ทดสอบก่อน Deploy (สำคัญมาก!):
    • ใส่ Sheet ID ของคุณที่บรรทัด DEBUG_SHEET_ID = '...'
    • เลือก testRun จาก dropdown (ด้านบน)
    • กด ▶ Run
    • ถ้ามี popup ขอ Permission → กด Review permissions → Allow
    • ดู Execution log ด้านล่าง → ต้องเห็น ✅
  6. Deploy → New deployment
    • Type: Web App
    • Execute as: Me
    • Who has access: Anyone
  7. กด Deploy → Copy URL ที่ได้มาใส่ด้านล่าง

⚠️ ข้อผิดพลาดที่พบบ่อย:
• ลืม Allow Permission ตอน testRun → script ทำงานไม่ได้
• ใช้ "Anyone with Google account" แทน "Anyone" → block คนอื่น
• Sheet ไม่ได้ Share หรือ Share แค่ "Restricted" → เปิดเป็น Viewer ได้เลย
• แก้ Code แล้วไม่ Deploy ใหม่ → URL เดิมยังรัน code เก่า

❓ EXIT_DD คืออะไร?
EXIT_DD = Emergency Exit เมื่อ Drawdown เกิน -6%

คือระบบป้องกันความเสี่ยง — ถ้ามูลค่าพอร์ตลดลงจาก Peak เกิน 6%
ระบบจะสั่ง ขายออกทั้งหมด → ย้ายเข้า Money_Market 100%

ตัวอย่าง: Peak = 130,000฿ → ถ้า Portfolio ลงมาที่ 122,200฿ (-6%) → EXIT

หลัง EXIT:
  • รอ 10 วันทำการ (Exit Cooldown)
  • รอให้ Signal กลับมาแข็งแกร่ง (score ≥ 0.55)
  • ถ้า EXIT 2ครั้งใน 30วัน → รอพักเพิ่มอีก (Extra Pause)
